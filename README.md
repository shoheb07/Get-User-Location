# Get-User-Location
Get User Location is a simple web application that uses the HTML5 Geolocation API to retrieve a user's current geographic location (latitude and longitude) through their browser.  When the user clicks the "Get My Location" button, the browser requests permission to access the device’s location. Once approved, the app displays:
Get-user-location
│── index.html
│── style.css
│── script.js
Features
Get real-time user location
Displays:
Latitude
Longitude
Error handling for:
Permission denied
Location unavailable
Timeout
Clean and responsive UI
